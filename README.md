# OTA

OTA Coding Test

## Installation and Running the application

System Requirements:

- Node 20.11.1

```bash
git clone git@github.com:kyleavedana/ota.git
cd ota
npm ci
npm start
```

set .env values

```text
PORT=3000 # Whatever port number you prefer
```

## Swagger

Check the `/api-docs` path to view Swagger
